XPath & Selenium Locator Cheat Sheet

This cheat sheet uses examples from your form to help you locate elements with both XPath and Selenium locator strategies.

# Type Description XPath Example Selenium Locator
1 Absolute XPath Full path from root /html/body/form/label[1]/input By.ID("input-name")
2 Relative XPath Starts from anywhere with // //input[@id='input-name'] By.ID("input-name")
3 Using Attributes Locate by attribute like name //input[@name='email'] By.NAME("email")
4 Text Content Match exact text //button[text()='Submit'] By.XPATH("//button[text()='Submit']")
5 Contains() Partial attribute match //input[contains(@id, 'name')] By.CSS_SELECTOR("input[id*='name']")
6 Starts-with() Match beginning of attribute //input[starts-with(@id, 'input-')] By.CSS_SELECTOR("input[id^='input-']")
7 AND/OR Multiple attribute conditions //input[@type='radio' and @value='female'] By.XPATH("//input[@type='radio' and @value='female']")
8 Position / Index Find element by order (//input[@type='radio'])[2] By.XPATH("(//input[@type='radio'])[2]")
9 DOM Traversal Find nested elements //div[@id='member-count-1']/input By.ID("input-member-1")
10 Wildcard * Match any tag with condition //*[@class='submit-button'] By.CLASS_NAME("submit-button")
